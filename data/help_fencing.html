Fencing Mechanism

SBD

Device list specifies the devices to use for exchanging sbd messages monitoring.

SBD options specifiy additional options for SBD start.

Delay SBD start specifies whether the node should pause for "msgwait" seconds on boot. This may be necessary if your cluster nodes reboot so fast that the other nodes are still waiting in the fence acknowledgement phase. This is an occasional issue with virtual machines.
